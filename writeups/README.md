## Writeup Template

When writing a report, use this structure:

```
## Title
[Clear description of the vulnerability]

## Summary
1-2 sentences: what is vulnerable, what can an attacker do.

## Severity
[Critical / High / Medium / Low / Informational]
CVSS Score: X.X

## Steps to Reproduce
1. Navigate to...
2. Enter the following payload...
3. Observe that...

## Impact
What an attacker can achieve by exploiting this.

## Proof of Concept
[Screenshot / video / code]

## Remediation
How to fix the issue.

## References
- OWASP: ...
- CVE: ...
```
