A 365-day structured roadmap, resource library, and progress log for going from AppSec Engineer to Master DevSecOps Engineer — covering Application Security, Cloud Security, CI/CD Pipeline Security, and Security Automation.

> **This is a tracking repo, not a trophy case.** Every phase below has a completion checkbox. If it's not checked, it's not done — no exceptions, no partial credit.

---

## 📌 Why This Repo Exists

Most "DevSecOps roadmap" repos on GitHub are read-once and abandoned. This one is different by design:

- Each phase has **shippable deliverables** (a tool, a writeup, a published finding, a working pipeline) — not just "studied X"
- Progress is logged weekly in [`roadmap/progress-log.md`](roadmap/progress-log.md)
- No phase is marked complete without a linked artifact (repo, blog post, HackerOne report, CVE, etc.)
